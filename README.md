# 汉字库
汉字库，收集汉字的拼音、注音、笔画、部首等等信息并以规范格式储存，用以开发使用。

创建本库目的为收集汉字的信息，暂时先通过爬虫的方式从 https://www.hanyuguoxue.com/ 网站上爬取了几万个汉字的信息。

汉字信息类型（括号里为举例）：
- [x] 拼音（qiū）
- [x] 注音（ㄑㄧㄡ）
- [x] 五笔（GJGG）
- [x] 部首（一部）
- [x] 结构（上下结构）
- [x] 笔画（5）
- [x] 部外笔画（4）
- [x] 笔画（21211）
- [x] 统一码（U+3400）
- [x] 繁体（孫）
- [x] 简体（孙）
- [] 收集词语，用以在生成拼音时防止多音字干扰，词语要按首中尾分类，以减少分析的数量
